# 复读机

众所周知，人类的三大本质之一：**复读机**

人类热爱玩梗跟风，虽然不明其意也要赶潮流的本质。

虽说复制粘贴过程中，只需要按两次键盘，但累计次数多了，手也会累的。

所以我写了一个复读机的小程序，可以让你在做复读机的同时，还能兼顾其它事情。
